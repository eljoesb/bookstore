
## Description

Simple App for aplicate test e2e

## Installation
```
$ docker-compose up
```
```bash
$ npm install
```

## Running the app

```bash
# development

$ npm run start

# watch mode

$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e

```
